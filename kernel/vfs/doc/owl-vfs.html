Introduction

Cette documentation à pour but de présenter le fonctionnement et l'utilisation du système de fichier virtuel de OWL. Elle décrit ses structures de données et son API.

Présentation générale

Le VFS est une couche d'abstraction du système d'exploitation qui permet d'accéder aux fichiers et données des périphériques de stockage de manière transparente quelque soit le périphérique ou le système de fichier, qu'ils soient locaux ou distants.

Le but du VFS d'OWL est de présenter une API compatible avec les appels POSIX.

OWL-VFS 0.1 se compose en premier d'une fonction d'initialisation qui monte le volume racine sur '/' et monte ensuite les autres volumes à partir du fichier /etc/fstab. Ensuite, le vfs propose des fonctions utilitaires pour (dé)monter les volumes, lire, écrire des fichiers ou des répertoires.
